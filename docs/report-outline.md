# Esquema del informe

El informe final se entrega como un PDF de dos páginas llamado
`TorresJ-MiniIdM.pdf`.

Se sugiere la siguiente estructura:

```text
Título: Infraestructura de Identidad Segura para la FIS
1. Arquitectura y nodos desplegados
2. Diseño de PKI, LDAP y Kerberos
3. Análisis de alta disponibilidad y seguridad
4. Resultados de pruebas y evaluación del rendimiento
5. URL del repositorio de GitHub
6. Declaración de asistencia externa
```

El nombre final del archivo usa únicamente caracteres ASCII. El PDF final no
se genera en este repositorio.
