# Report outline

The final report must be a two page PDF named `TorresJ-MiniIdM.pdf`.

Suggested structure:

```text
Title: Infraestructura de Identidad Segura para la FIS
1. Architecture and deployed nodes
2. PKI, LDAP and Kerberos design
3. High availability and security analysis
4. Test results and performance evaluation
5. GitHub repository URL
6. External assistance disclosure
```

Use only ASCII characters in the final file name. The final PDF is not generated by this repository step.
