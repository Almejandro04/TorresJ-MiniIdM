# Resultados

Este directorio almacena la evidencia generada por las pruebas de laboratorio.

| Directorio | Contenido |
|---|---|
| tables | Métricas CSV de replicación, conmutación por error, TLS, carga y fallos de nodo |
| figures | Capturas o gráficos usados en el informe |
| raw | Capturas temporales y respaldos LDAP, ignorados por Git |

En `results/` no se almacenan contraseñas, keytabs, claves privadas ni datos
de producción.
